## Screenshot Guidelines

When adding screenshots:
1. Use PNG format for static images
2. Use GIF format for animations (max 5MB)
3. Recommended resolution: 1920x1080 or 1280x720
4. Highlight key features with annotations if needed
5. Include both light and dark theme versions if applicable

## Example Naming Convention

```
feature-name-view.png
feature-name-action.gif
feature-name-error.png
```
